# Async Development Skill

This skill enables asynchronous development workflows where Claude can work on tasks in the background.

## Usage

When invoked, this skill allows Claude to:
- Run long-running tasks without blocking the conversation
- Queue multiple tasks for sequential execution
- Provide progress updates asynchronously

## Triggers

- User mentions "background task"
- User asks to "run this while I do something else"
- Explicit invocation with `/skill async-dev`
